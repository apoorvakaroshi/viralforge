# ⚡ ViralForge

> **Create viral marketing ideas. Get real votes. Rise to the top.**

A public scoreboard where digital marketing ideas compete for real attention. No ads. No tricks. Just your creativity vs the world.

🔗 **Live Demo:** [viralforge-apoorva.netlify.app](https://viralforge-apoorva.netlify.app)

---

## 🎯 What is ViralForge?

ViralForge is a platform built for the **ConnectsBlue Digital Marketing Challenge**. Creators post their best digital marketing concept — a viral hook, meme ad, short video idea, or bold campaign — and the internet votes. The strongest ideas rise to the top.

No followers needed. No paid promotion. Just pure creative power.

---

## ✨ Features

- 🪝 **4 Campaign Types** — Viral Hook, Meme Ad, Video Concept, Bold Campaign
- 🔥 **Real-time Voting** — Community votes decide who rises
- 🏆 **Public Leaderboard** — Live rankings updated instantly
- 🧬 **Virality DNA** — Creativity score, emotion detection & momentum tracking
- 🎯 **Creative Brief Board** — Daily briefs to spark campaign ideas
- ⚡ **Trending Triggers** — See what emotional hooks are working right now
- 💡 **Idea Spark Generator** — AI-style prompts to beat creative block
- 📊 **Analytics Dashboard** — Engagement insights per campaign
- 🎨 **Particle Animation** — Interactive canvas background
- 📱 **Fully Responsive** — Works on mobile and desktop

---

## 🛠️ Built With

- **HTML5** — Structure
- **CSS3** — Styling, animations, dark mode
- **Vanilla JavaScript** — All logic, no frameworks
- **Canvas API** — Particle system
- **Intersection Observer API** — Scroll animations

## 📁 Project Structure

```
viralforge/
├── index.html       # Main app structure

├── particles.js     # Interactive particle background
└── manifest.json    # PWA manifest
```

## 🚀 Run Locally

No build step needed — it's pure HTML/CSS/JS.

```bash
# Clone the repo
git clone https://github.com/apoorvakaroshi/viralforge.git

# Open in browser
cd viralforge
# Double-click index.html OR use Live Server in VS Code
```
